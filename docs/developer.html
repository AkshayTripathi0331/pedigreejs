pedigreejs

A web tool for building and displaying pedigrees.

Developer Notes

The pedigree data is stored in the browser window as HTML local or session storage or as an JS array. This is defined by the 'store_type' configuration settings. The pedigree data displayed can be accessed by calling the JavaScript routine: 

var pedigree = pedcache.current(opts);

This returns the pedigree data in JSON format. A search routine, for example, can then be implemented to interogate this.
