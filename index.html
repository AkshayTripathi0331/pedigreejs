pedigreejs

A web tool for building pedigrees.

pedigreejs is a JavaScript library that uses d3 to layout the hierarchical structure and draw the individual nodes using standard pedigree nomenclature. An example is given below.

Individuals can be added to the pedigree by clicking on another individual and adding a partner, parent or sibling. There are 'undo' , 'redo' and 'reset' buttons at the top and a button to enable full screen editing .
